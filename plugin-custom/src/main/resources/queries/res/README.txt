Este diretório possui os arquivos de internacionalização de mensagens e textos, apresentados nas consultas dinâmicas.

O arquivo com os textos padrão está no projeto "auster:dware-console-actions", no diretório src/main/resources/queries/res. Caso seja
necessário customizar mensagens existentes, ou acrescentar novos textos, o arquivo original deve ser copiado para este diretório, e as 
novas mensagens acrescentadas.

IMPORANTE: A nomenclatura do arquivo deve ser mantiva : "locale" + <string de locale> + ".properties"
